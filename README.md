# Projet 7 — Prévision de la demande de vélos partagés

Dépôt : [github.com/eman-mekkaoui/explodata](https://github.com/eman-mekkaoui/explodata)

Notebook de régression sur le [Bike Sharing Dataset](https://archive.ics.uci.edu/ml/datasets/bike+sharing+dataset) (UCI).

## Installation (sans venv)

```bash
pip install -r requirements.txt
python -m ipykernel install --user --name explodata
```

## Lancer le notebook

Dans VS Code / Cursor : sélectionner le kernel **Python 3** (ou **explodata** après l'étape ci-dessus), puis ouvrir `projet7_bike_sharing.ipynb`.

En ligne de commande :

```bash
jupyter notebook projet7_bike_sharing.ipynb
```

Les données se téléchargent automatiquement au bloc 2 si elles ne sont pas déjà présentes.
